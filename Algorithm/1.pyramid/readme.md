# 피라미드
난이도 : 중하
예상 소요 시간: 30분 이상
그 유명한 (*)로 피라미드를 쌓는 문제 입니다. 복습겸 이 문제를 풀어 봅시다.

## 문제

표준 입력으로 삼각형의 높이가 입력됩니다. 입력된 높이만큼 산 모양으로 별을 출력하는 프로그램을 만드세요(input에서 안내 문자열은 출력하지 않아야 합니다). <br>
이때 출력 결과는 예제와 정확히 일치해야 합니다.

---
## 예시

### Input 1
~~~
3
~~~

### Output1
~~~
  *
 ***
*****
~~~
---

### Input2
~~~
5
~~~
### Output2
~~~
    *
   ***
  *****
 *******
*********
~~~
